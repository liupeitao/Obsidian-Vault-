## Django 学习计划

### 第一周

* Day 1: 入门

  * 安装 Django
  * 创建一个新项目
  * 启动开发服务器
* Day 2: 视图和 URL

  * 创建一个视图函数
  * 创建一个 URL 映射到该视图
* Day 3: 模板

  * 创建一个简单的模板
  * 在视图中使用模板
* Day 4: 静态文件

  * 添加 CSS 和 JavaScript 文件到你的项目中
  * 在模板中加载静态文件
* Day 5: 数据库

  * 连接到数据库
  * 创建一个模型类

### 第二周

* Day 6: 查询数据

  * 使用 Django ORM 查询数据
  * 在视图中展示数据
* Day 7: 表单

  * 创建一个表单类
  * 在视图中处理表单数据
* Day 8: 用户认证

  * 添加用户认证功能到你的项目中
  * 创建登录和注册视图
* Day 9: 中间件

  * 创建自定义中间件
  * 在视图中使用中间件
* Day 10: 部署

  * 将你的 Django 应用程序部署到云端服务器上

希望这份计划能够帮助您开始学习 Django！
